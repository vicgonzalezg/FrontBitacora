1.-Borrar entorno virtual
2.-Instalar gtk3-runtime-3.24.29-2021-04-29-ts-win64.exe(Instalacion por defecto)
3.-Path variable usuario C:\Program Files\GTK3-Runtime Win64\bin debe quedar de las primeras
4.-Path varaible sistema C:\Program Files\GTK3-Runtime Win64\bin debe quedar de las primeras
5.-Crear entorno virtual -> python -m venv myvenv
6.-Activar entorno virtual -> myvenv\Scripts\activate
7.-Instalar PIP -> python -m pip install --upgrade pip
8.-Instalar requirements -> pip install -r requirements.txt
